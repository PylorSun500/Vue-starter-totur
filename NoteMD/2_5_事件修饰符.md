### 阻止事件冒泡 `.stop`

[ed_stop.vue](../Code/vueint/src/components/2_5_event_decoration/ed_stop.vue)

`<a><b>123</b></a>`

冒泡是 DOM 的一个特性，表现为事件从内向外传递。
这样的结构中，b 标签的行为会向上触发 a 标签。使用 `.stop` 可以阻止向上触发。
放在嵌套 DOM 的任意一层时，事件由内向外冒泡，冒泡到此时，停止。

### 仅自身触发 `.self`

[ed_self.vue](../Code/vueint/src/components/2_5_event_decoration/ed_self.vue)

标题这么老长，作用其实就是：

```
只有点到这个元素本体时，事件才执行；如果是点到其他的元素，这个元素不会执行。
```

### 监听滚动 `passive`

添加 `.passive` 后，会优先响应滚动事件，而非其回调函数。
如果再解释地严谨些，可以理解为监听器不会调用 `preventDefault()` ，这样一来，浏览器不需要等待回调函数执行后再决定是否滚动，减少了等待时间，会让滚动更流畅。

常用于移动端，提升移动端性能。

### 捕获按键

[ed_button.vue](../Code/vueint/src/components/2_5_event_decoration/ed_button.vue)
[ed_mouse.vue](../Code/vueint/src/components/2_5_event_decoration/ed_mouse.vue)

捕获按键分两部分：**键盘修饰键** 和 **鼠标键** 。
直接通过 `.name` 来修饰。

目前可以识别的按键有：

- 键盘修饰键

    - `.enter` Enter / Return
    - `.esc` Esc
    - `.tab` Tab
    - `.delete` Delete / Backspace
    - `.ctrl` Ctrl / control
    - `.alt` Alt / option
    - `.shift` Shift
    - `.meta` Windows Logo / command / Super / Meta

- 鼠标键

    - `.left` 鼠标左键
    - `.right` 鼠标右键
    - `.middle` 鼠标中键点击
