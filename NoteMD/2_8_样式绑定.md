# 样式绑定

通过绑定 `class` 和 `style` 来实现样式绑定，让网页元素的样式（颜色、大小、位置等）能够根据数据自动变化。
使用 `v-bind` 来实现动态绑定属性。这里我们只讨论使用它来绑定样式。
至于 `v-bind` 的更多用法，可能会在后续提到。

## 绑定 `class` 属性

使用 `v-bind:class` 来操作 class 属性。

### 将 class 属性值绑定为字符串

这种方案类似于为 css 类名提供一个 **别名** 。
代码行文思路如下：

**1. 定义样式：**

在 `<style>` 标签中，定义需要的 CSS 类：

```css
.box {
    border: 1px solid black;
}
```

**2. 声明响应式数据**

声明一个响应式变量（例如 className），并将其初始值设置为你想应用的 CSS 类名。

```javascript
const className = 'box'
```

**3. 最后，在模板中进行绑定**

```html
<div v-bind:class = "className">ABC</div>
```

这个时候，`box` 的别名就被更换成了 `className` 。当然，作为响应式数据，我们也可以在代码中直接修改 `className` 的值，来达到在一定条件下直接更换 `css` 的效果。
