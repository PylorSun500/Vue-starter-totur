# 样式绑定

通过绑定 `class` 和 `style` 来实现样式绑定，让网页元素的样式（颜色、大小、位置等）能够根据数据自动变化。
使用 `v-bind` 来实现动态绑定属性。这里我们只讨论使用它来绑定样式。
至于 `v-bind` 的更多用法，可能会在后续提到。

## 绑定 `class` 属性

使用 `v-bind:class` 来操作 class 属性。

### 将 class 属性值绑定为字符串

如：
在模板文件中，设定：

```html
<div v-bind:class = "className">ABC</div>
```
```css
.box {
    border: 1px solid black;
}
```

这时，`class` 属性被绑定为字符串 `className`
然后就可以在 `<script setup>` 中将 `className` 赋值给一个具体的样式类，如 `box` ：

```javascript
const className = 'box'
```
