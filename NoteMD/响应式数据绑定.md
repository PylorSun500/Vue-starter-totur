# Vue 3 响应式数据绑定

Vue 3 提供了多种创建响应式数据的方式。以下是常用的响应式函数及其用法：

## 1. ref() 函数

将基本数据类型转换为响应式**数据**。

### 基本用法
```javascript
响应式数据 = ref(初始值)
```

### 修改数据
```javascript
响应式数据.value = 新值
```

### 示例代码
```vue
<template>{{ msg }}</template>
<script setup>
import {ref} from 'vue'
const msg = ref('会当凌绝顶，一览众山小')
setTimeout(() => {
    msg.value = '契而不舍，金石可镂'
}, 2000)
</script>
```

## 2. reactive() 函数

将一个对象或数组转换为响应式对象。

### 基本用法
```javascript
响应式对象 = reactive(初始对象)
```

### 示例代码
```vue
<template>{{ object.msg }}</template>
<script setup>
import {reactive} from 'vue'
const object = reactive({msg: '不畏浮云遮望眼，自缘身在最高层'})
setTimeout(() => {
    object.msg = '欲穷千里目，更上一层楼'
}, 2000)
</script>
```

## 3. toRef() 函数

将响应式对象的某个属性转换为独立的响应式数据引用。

### 基本用法
```javascript
响应式数据 = toRef(响应式对象, '属性名')
```

### 示例代码
```vue
<template>
<div>msg = {{ msg }}</div>
<div>obj.msg = {{ obj.msg }}</div>
</template>

<script setup>
import {reactive, toRef} from 'vue'
const obj = reactive({msg: '黑发不知勤学早，白首方悔读书迟'})
const msg = toRef(obj, 'msg')
setTimeout(() => {
    msg.value = '少壮不努力，老大徒伤悲'
}, 2000)
</script>
```

## 4. toRefs() 函数

将响应式对象的所有属性转换为独立的响应式数据引用。

### 基本用法
```javascript
所有属性组成的对象 = toRefs(响应式对象)
```

### 示例代码
```vue
<template>
<div>msg = {{ msg }}</div>
<div>obj.msg = {{ obj.msg }}</div>
</template>

<script setup>
import {reactive, toRefs} from 'vue'
const obj = reactive({msg: '盛年不重来，一日难再晨'})
let {msg} = toRefs(obj)
setTimeout(() => {
    msg.value = '及时当勉励，岁月不饶人'
}, 2000)
</script>
```

## 函数对比表

| 函数 | 适用类型 | 返回值 | 修改方式 | 特点 |
|------|----------|--------|----------|------|
| `ref()` | 基本类型 | `Ref` 对象 | `.value` | 适用于基本数据类型，返回包装对象 |
| `reactive()` | 对象/数组 | 响应式代理对象 | 直接属性访问 | 适用于对象和数组，深度响应式 |
| `toRef()` | 对象属性 | `Ref` 对象 | `.value` | 创建对象属性的响应式引用 |
| `toRefs()` | 整个对象 | 包含多个 `Ref` 的对象 | 每个属性 `.value` | 解构响应式对象时保持响应性 |

---

## 使用建议

1. **基本类型** → 使用 `ref()`
2. **对象/数组** → 使用 `reactive()`
3. **需要解构对象属性** → 使用 `toRefs()`
4. **需要单独引用对象属性** → 使用 `toRef()`

## 注意事项

- `ref()` 返回的对象需要通过 `.value` 访问和修改值
- `reactive()` 返回的代理对象可以直接访问属性
- `toRef()` 和 `toRefs()` 主要用于保持响应式对象的属性响应性
- 在模板中，`ref()` 和 `reactive()` 都会自动解包，无需使用 `.value`
